# C-004 — ephemeral tagging + default-exclude test runs

status: done
deps: none
implements: FR5

## Scope

Tag throwaway/test runs and exclude them from analytics by default. In flow.sh `_log_event`,
compute `ephemeral` (true if project root is under the system temp dir OR basename matches
`^tmp\.`) and write it to BOTH the full and compact JSON lines. In flow_harness.py: add an
`ephemeral` column to `usage_event` (default 0, tolerate old lines without it); default-exclude
ephemeral rows in `rollup`/`usage` with a read-time fallback (`project LIKE 'tmp.%'`); add
`--include-ephemeral` to include them.

- `skills/flow/runner/flow.sh` (`_log_event` ephemeral detection + both printf lines)
- `skills/flow/harness/flow_harness.py` (schema column, rollup ingest, usage filter, arg)
