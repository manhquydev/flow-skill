# C-001 — usage --global forwards --global to the rollup step

status: done
deps: none
implements: FR1

## Scope

In `cmd_usage` (skills/flow/runner/flow.sh), forward `--global` to the preliminary `rollup`
call when the args contain `--global`, so the device-global JSONL is ingested into the
`usage_event` mirror before `usage --global` queries it. Today the rollup runs without
`--global`, so `usage --global` always reports "no events".

## Allowed files

- `skills/flow/runner/flow.sh` (only `cmd_usage`)
- `tests/` (add a regression test)
