# C-002 — cycle_id stamped at all entry points (assess + lazy-if-absent)

status: done
deps: none
implements: FR2

## Scope

Make every logged event carry a non-empty `cycle_id` on brownfield/assess-first projects.
Reuse the existing `CYCLE_FILE` mechanism (format `epoch-host`). Add a `_ensure_cycle`
helper that stamps `CYCLE_FILE` if absent, call it from `cmd_assess`, and have `_log_event`
lazily stamp/read it so pre-existing projects also get one. Do NOT add uuid/json deps.

## Allowed files

- `skills/flow/runner/flow.sh` (cycle helper, `cmd_assess`, `_log_event` read path)
- `tests/` (regression test)
